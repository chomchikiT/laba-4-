# Матрица приоритизации электронных анкет

Практическая работа по JavaScript — интерактивная матрица приоритизации электронных анкет.

## Что делает приложение

- Показывает список анкет в таблице с рассчитанными оценками
- Распределяет анкеты по 4 зонам матрицы
- Можно добавлять новые анкеты через форму
- Можно удалять анкеты
- Сортировка по столбцам (клик по заголовку)
- Фильтр по зонам матрицы
- Данные сохраняются в браузере (localStorage)
- Экспорт в JSON файл

## Как запустить

Скачать файлы и открыть index.html в браузере. Ничего устанавливать не нужно.

## Файлы

- `index.html` — разметка страницы
- `style.css` — стили
- `app.js` — логика на JavaScript

## Технологии

HTML, CSS, JavaScript (vanilla)
